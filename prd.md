# 내명함 랜딩페이지(rv2) prd

## 1. 목적
on/off line 모임이나 강의시 종이명함 대신 보여줄 온라인명함.
갱신된 내용을 바로바로 update시켜 상대에게 보여줄 수 있는 것이 목적임.

## 2. 사용자 (페르소나)
1. 이 페이지를 '전달하는 사람(나)' :
  - 한덕환 / small N잡러 /유튜브작가·AI컨설턴트·길여행가
  - 외부 강연 자리, 컨설팅고객과 대면하는 자리
2. 이 페이지를 '받아 보는 사람(상대방)' :
  - 어떤 기기로 보는가 : 주로 스마트폰 (모임 자리에서 바로 열람)
  - 얼마나 오래 볼 것 같은가 : 10~20초
  - 무엇을 알고 싶어 하는가 : 한덕환이 지금은 무엇을 하는 사람이고, 예전엔 어떤 일을 했던 사람인지, 연락처와 소통 링크 공유

## 3. 기능
### 필수 기능
1. 페이지 상단에 원형 프로필 사진(profile_po.jpg)가 표시됨.
2. 원형 프로필 사진의 하단 1/3 위치에서 부터 위로 배경사진(bg_po.jpg)가 채워짐 (프로필 사진의 공간을 제외한 여백)
3. 글자체: 깔끔한 산세리프체
4. 스타일: 그림자 없는 깔끔한 2D 그림, 깔끔한 일러스트레이션 (순백색 배경 #FFFFFF)
5. 원형 프로필 사진 아래, 첫째 줄에 'small N잡러'(본문 글자크기와 동일, regular체)를 나타내고, 그 옆에 '한덕환'을 (본문글자크기의 1.5배로, bold체)표시한다.
6. 둘째 줄에 '《이제는. 좋아하는 일만 하는 사람》' 표시(본문 글자크기와 동일, bold체)
7. 세째줄 부터 한 줄씩 아래 현재경력을 표시한다. 각 경력들은 왼쪽 정렬되도록 배치한다.
지금.유튜브작가/@혼자걷다보면(2025서울대표 인플루언서 100인)
지금.소상공인 AI컨설턴트
지금.전문강사(서울시·대기업·공기관/중장년의삶)
지금.길여행가/둘레길강사
지금.브런치작가/투빈대디
직전.국립공원안전산행지원단
직전.구로구청 청년일자리멘토
8. 다음 줄에 '《그전엔. 직장인 30년》' (본문 글자크기와 동일, bold체)를 표시하고 아래 줄에 과거경력을 한 줄로 표시한다. 현재경력과 왼쪽 정렬되게 배치한다.
9. 다음 줄에 '큰기업근무 15년. 작은기업경영 15년' 표시 (본문 글자크기와 동일, regular체)

10. 한줄의 빈칸을 두고, 다음 줄에 '이메일' 버튼(hanmokhae@gmail.com)을 만들고, 누르면 기본 메일 앱이이 열리도록 한다.
10. '연동사이트'를 본문크기 regular, 하늘색 글자색으로 표시하고, 
그 아래에 '@혼자걷다보면(유튜브)'(링크주소: https://www.youtube.com/@혼자걷다보면) 정사각형 버튼을 만들고, 그 버튼을 누르면 유튜브 해당 주소로 자동 연동되도록 한다. 
11. 그 오른쪽에 나란히 '투빈대디(브런치)'(링크주소: https://brunch.co.kr/@2beendaddy) 정사작형 버튼을 만들고, 그 버튼을 누르면 브런치 해당주소로 자동연동되도록 한다.

## 4. 제약
### 의도적으로 하지 않는 것
- 회원가입·로그인 기능은 넣지 않습니다.
- 방문자 통계·조회수·댓글·방명록은 넣지 않습니다.
- 한국어 단일로 만듭니다.

### 환경적 한계
- HTML·CSS만 사용합니다. 서버 기능·데이터베이스는 쓰지 않습니다.
- 페이지는 크롬 브라우저와 스마트폰(세로 화면) 모두에서 읽혀야 합니다.
